The PWMPortEnum Enum

The PWMPortEnum enum defines possible PWM ports on the Raspberry Pi.

 

Source Code

The source code for this class is available online for download and also in browseable form.

 

Syntax

    public enum PWMPortEnum
    {
        PWM_NONE,
        PWM_0,       // output pin depends on dt_overlay
                     //    GPIO_18, header Pin12 by default
        PWM_1,       // output pin depends on dt_overlay       
                     //    GPIO_19, header Pin35 by default
    }
